# System Requirements

| Sr.# | Requirement/Description | Stakeholder | Traceable User Requirements |
|------|------------------------|-------------|----------------------------|
| S1 | The system must have a user database to manage user credentials, providing user access based on their user role of customer, staff, manager, or owner. | Owner, Manager, Staff, Customer | U6, U15, U16, U17, U18 |
| S2 | The system must include inventory management, providing real time information on and the ability to edit stock | Owner, Manager, Staff | U1, U2, U8 |
| S3 | The system must include a POS (point of sale) system that allows employees to process transactions and provide receipts, including different payment types, as well as multiple payments at once while also allowing customers to save their reward points. | Staff, Customer | U4, U6, U18 |
| S4 | The system must have role based security based on logged in user credentials. | Owner, Staff, Customer, Manager | U15, U16, U17 |
| S5 | The system must include a menu management system which updates the pricing and display of available menu items for the owner and manager to use | Manager, Owner, Customer | U10, U11, U14 |
| S6 | The system must include a reservation system which enables staff and management to book seatings at specific times for anonymous and registered guest. | Owner, Staff | U19 |
| S7 | The system must save transactions and expenses from other parts of the system for analytical reports. | Owner | U9, U12, U13 |
| S8 | The system must allow multiple points of access, through which different users of different credentials can access information. | Staff, Manager, Owner | U15, U16, U17 |
| S9 | The system must include an IT department interface used to monitor the system for troubleshooting and updates. | IT User | U7 |
| S10 | The system will provide managers access to modify and check employee schedules, such as clock-in times, as well as payment while providing owner credentials access to expense and transaction reports. | Owner, Manager | U3, U5, U9, U12 |
