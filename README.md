# Small Food Business Management System

**Class Project** - COP 4331 (Software Engineering)  
**Fall 2025**

A desktop application for managing small food service establishments (cafes, food trucks, small restaurants).

##  What This System Does

- **Employee Management** - Payroll, scheduling, time tracking
- **Inventory Control** - Stock tracking with low-stock alerts
- **Point of Sale (POS)** - Process sales with multiple payment types
- **Menu Management** - Add/edit menu items
- **Customer Loyalty** - Points and rewards system
- **Reports** - Sales, expenses, profit analysis
- **Reservations** - Optional booking system

##  Tech Stack

**Frontend:** Electron + Vue.js + Tailwind CSS  
**Backend:** Node.js + PostgreSQL  
**Tools:** Sequelize (ORM), Chart.js, PDFKit, ExcelJS
